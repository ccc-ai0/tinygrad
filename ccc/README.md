karpathy 的 micrograd 用很精簡的方式做了反傳遞梯度引擎

但是只做到 MLP 的範例就停了

沒有 CNN/ RNN / Transformer / Diffusion 等

然後有人做了一個介於 micrograd 與 pytorch 之間的小型深度學習框架，稱為 tinygrad 

看來這個專案非常值得研究，或許可以成為 micrograd 的進階教材

該專案可以跑 YOLO / LLAMA 與 Stable Diffusion ...
